# baixar_videos_grupo_telegram
Aplicação que baixa vídeos de um grupo ou canal do Telegram, organizando os vídeos automaticamente em pastas conforme a hierarquia textual da mensagem.
Esta aplicação usa as bibliotecas extras: **Telethon**, **tqdm**, **python-dateutil** e **aiofiles**.

O **Telethon** é uma biblioteca Python para interagir com a API do Telegram. Ela permite criar clientes Telegram completos, controlar contas, bots, canais, grupos e enviar/receber mensagens programaticamente.

Para instalar o pacote **Telethon**, use o comando: `pip install `telethon`.

O **tqdm** é uma biblioteca que serve para exibir barras de progresso elegantes, rápidas e simples em loops no Python.

Para instalar o pacote **tqdm**, use o comando: `pip install tqdm`.

O **python-dateutil** é uma biblioteca que serve para trabalhar com datas e horários de forma muito mais avançada e flexível do que o módulo padrão _datetime_ do Python.
Ela é especialmente usada para parsear (converter strings em objetos datetime) e para manipulação inteligente de datas.

Para instalar o pacote **python-dateutil**, use o comando: `pip install python-dateutil`.

O **aiofiles** é uma biblioteca que serve para realizar operações de leitura e escrita de arquivos de forma assíncrona em Python.
Ela é usada principalmente em aplicações _async/await_, onde operações de I/O bloqueantes (como abrir, ler e escrever arquivos) poderiam travar o loop de eventos.

Para instalar o pacote **aiofiles**, use o comando: `pip install aiofiles`.
